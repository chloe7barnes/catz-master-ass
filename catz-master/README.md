![Sreenshot](SCREENSHOT.png)

## Development Setup

Before you do anything, you&rsquo;ll need to have [Git](https://git-scm.com/) and [Node](https://nodejs.org/en/) installed.

```shell
# clone
git clone https://github.com/cadejscroggins/catz
cd catz

# install
npm i -g yarn && yarn

# serve
yarn start
```
